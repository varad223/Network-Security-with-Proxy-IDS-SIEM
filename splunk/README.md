# SIEM – Splunk Integration

## Purpose
Splunk provides **centralized visibility** into firewall events and IDS/IPS alerts.

All security events are collected, indexed, and analyzed in one place.

---

## Logs Collected
- Suricata alerts
- Firewall events
- Network activity
- Blocked IP addresses

---

## Files
- `inputs.conf` – Log input configuration
- `forwarder_setup.md` – Universal Forwarder setup
- `splunk_server_setup.md` – SIEM server configuration
