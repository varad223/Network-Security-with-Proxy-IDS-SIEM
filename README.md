# Network-Security-with-Proxy-IDS-SIEM
Designed and implemented secure enterprise-grade network on Debian Linux/VirtualBox. Built multi-zone firewall (LAN, DMZ, WAN, VPN) via iptables, deployed web &amp; Squid proxy in DMZ, enabled OpenVPN remote access, integrated Suricata IDS/IPS, centralized SIEM monitoring, enforced NAT, strict access controls &amp; industry-best security policies.
